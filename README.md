**Guess the Number Game**
A simple, interactive command-line game built with Python.

**Description**

The computer picks a random number between 1 and 20, and the player has 6 attempts to guess it. After each guess, the game tells the player if their guess was too high or too low.

**How to Run**

1.  Ensure you have Python 3 installed.
2.  Clone the repository:
    git clone [suspicious link removed]
3.  Navigate to the folder:
    cd guess-the-number-python
4.  Run the script:
    python3 guess_the_number.py

**Features
**
  - Random number generation using the random module.
  - User input handling.
  - Conditional logic for "too high/low" feedback.
  - Maximum of 6 attempts.
